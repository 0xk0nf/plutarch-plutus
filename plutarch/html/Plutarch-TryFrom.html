PTryFrom a b represents a subtyping relationship between a and b, and a way to go from a to b. Laws: - (punsafeCoerce . fst) $ tcont (ptryFrom x) ≡ pure x

type family PSubtype (a :: PType) (b :: PType) :: Constraint where ... Source #

PSubtype a b constitutes a subtyping relation between a and b. This concretely means that `(x :: Term s b) -> punsafeCoerce x :: Term s a` is legal and sound.

You can not make an instance for this yourself. You must use the PInner type family of PlutusType to get this instance.

Caveat: Only PInner a POpaque is considered unfortunately, as otherwise getting GHC to figure out the relation with multiple supertypes is quite hard.

Subtyping is transitive.
